Comment utiliser ce module

Ce module génère des scripts robocopy pour automatiser l'archivage de fichiers vers le cold storage réseau.

1Téléchargez le modèle CSV et remplissez-le avec vos chemins source et destination.
2Importez le CSV via drag & drop ou en cliquant sur la zone de dépôt.
3Configurez les options robocopy : flags, log, exclusions.
4Prévisualisez les commandes dans le tableau, puis exportez en .cmd ou .xml.

Les chemins UNC générés sont compatibles avec les serveurs srvrbxassufp01 et srvrbxassufp01cold.
